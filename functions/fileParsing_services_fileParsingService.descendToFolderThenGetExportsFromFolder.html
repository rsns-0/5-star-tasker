Function descendToFolderThenGetExportsFromFolder

  • descendToFolderThenGetExportsFromFolder<TExportModel>(folderName, startPosition, options?): Promise<TExportModel[]>

  • Starts from the start position and performs bfs to find the first folder with the specified name. Afterwards, retrieves all exports from files within the folder.

    Type Parameters

    • TExportModel = any

      The interface representing the export object from a module.

    Parameters

    • folderName: string

      The name of the folder to search for files in.

    • startPosition: string

      The starting position of the search. Due to limitations, cannot create a default. Pass in __dirname to start from the current directory.

    • options: {
          fileFilter: undefined | ((fullFilePath) => boolean);
          recursive: undefined | boolean;
      } = {}

      An optional object containing the following properties:

      • fileFilter: A function that determines whether a file should be included in the search. Defaults to .ts and .js files.
      • recursive: A boolean indicating whether to search recursively through subfolders for file exports after reaching the target folder. Defaults to false.
      • fileFilter: undefined | ((fullFilePath) => boolean)
      • recursive: undefined | boolean

    Returns Promise<TExportModel[]>

    A promise that resolves to an array of exported models.

    Throws

    • If the folder is not found or if the starting directory does not exist.
